Lazos de Control y Automatización

Estrategias de control implementadas en el PLC para operación autónoma del sistema.

🔄 Lazos Principales

  • Control de presión SWRO: PID sobre VFD de bomba alta presión para mantener presión de operación objetivo (60 bar nominal).
  • Control de nivel tanque producto: ON/OFF con histéresis. Inicia producción cuando nivel < 30%, detiene cuando > 90%.
  • Control de pH: Dosificación proporcional de NaOH con medición en línea de pH. Setpoint: 7.8 ± 0.3.
  • Secuencia de arranque: Apertura válvulas → purga de aire → arranque bomba captación → pre-tratamiento → arranque BAP → rampa de presión.
  • Parada de emergencia: Cierre inmediato de válvulas, parada de bombas, alarma sonora y visual, notificación remota.

SCADA

AQUA-CLOUD: Monitoreo Remoto

Plataforma web para supervisión remota del sistema desde cualquier dispositivo con conexión a internet.

📱

Dashboard en Tiempo Real

Visualización de variables críticas: TDS producto, presión SWRO, nivel tanques, producción diaria, estado del sistema y alarmas activas. Acceso desde navegador web o app móvil.

📈

Históricos y Tendencias

Gráficos de tendencia de todas las variables del proceso con resolución de 1 minuto. Descarga de datos en CSV para análisis y reportes. Almacenamiento local y en la nube.

🔔

Alertas y Notificaciones

Configuración de umbrales de alarma para cada variable. Notificaciones push, SMS y correo electrónico para eventos críticos: presión alta, TDS elevado, nivel bajo, falla de bomba.

🤖

AQUA-AI: Mantenimiento Predictivo

Algoritmo de machine learning que analiza tendencias de presión diferencial, calidad de permeado y horas de operación para predecir limpieza de membranas (CIP) y mantenimiento preventivo.
