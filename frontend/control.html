Control y SCADA

Automatización industrial, monitoreo en tiempo real y gestión inteligente

Controlador

PLC + HMI Local

Controlador Lógico Programable (PLC) con pantalla HMI (Human Machine Interface) para operación local del sistema, secuencias automáticas y alarmas.

  • PLC: Compacto, 16-32 I/O digitales/analógicos
  • HMI: Pantalla táctil 7-10 pulgadas
  • Comunicación: Modbus RTU/TCP, Ethernet
  • Programación: Ladder, bloques de función
  • Funciones: Arranque/parada automática, alarmas, data logging
  • Costo estimado: USD $1,800
🤖 Foto: PLC con HMI táctil instalado en panel de control
Distribución

Tablero Eléctrico + Arrancadores

Tablero de distribución eléctrica con arrancadores suaves (soft starters) para bombas de alta potencia, protecciones y señalización.

  • Tablero: NEMA 4X, acero inoxidable
  • Arrancadores: Soft starter para bomba HP (reduce arranque)
  • Variadores: VFD para bomba de captación
  • Protecciones: Guardamotores, relés térmicos
  • Costo estimado: USD $1,500
⚡ Foto: Tablero eléctrico con arrancadores y protecciones
Instrumentación

Sensores Generales

Red de sensores para monitoreo de niveles, presiones y temperaturas en todo el sistema, conectados al PLC para control automático.

  • Nivel tanques: Sensores ultrasónicos o de boya
  • Presión: Transmisores 4-20 mA
  • Temperatura: RTD PT100
  • Flujo: Caudalímetros de turbina
  • Costo estimado: USD $700
📡 Foto: Sensores de nivel, presión y flujo instalados
Software

AQUA-CLOUD Dashboard

Dashboard web para monitoreo remoto del sistema desde cualquier dispositivo con internet. Integrado con el backend Python ya desplegado en Render.

  • Monitoreo en tiempo real: Presión, flujo, conductividad, pH
  • Alarmas: Notificaciones por email/SMS
  • Historial: Tendencias y reportes
  • Control remoto: Arranque/parada desde la nube
  • Multiusuario: Roles de operador y administrador
